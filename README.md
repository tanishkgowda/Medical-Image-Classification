# Medical Image Classification Using SVM, CNN, and XGBoost

## Description

This project investigates the application of machine learning (ML), deep learning (DL), and hybrid ML-DL approaches for medical image classification using Brain MRI and Chest X-Ray datasets. The objective is to classify brain tumor types from MRI scans and detect pneumonia from chest X-ray images. Multiple classification techniques, including SVM + PCA, CNN (MobileNet Transfer Learning), XGBoost, PCA + XGBoost, and VGG16 + XGBoost, are evaluated and compared across different training data proportions. The project explores the impact of feature extraction, dimensionality reduction, transfer learning, and ensemble learning on classification performance, providing a comprehensive comparison of traditional machine learning, deep learning, and hybrid approaches for automated medical image analysis.

## Datasets Used

### Brain MRI Dataset

* Source: https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri
* Classes:

  * Glioma Tumor
  * Meningioma Tumor
  * Pituitary Tumor
  * No Tumor

### Chest X-Ray Dataset

* Source: https://www.kaggle.com/code/khanfashee/medical-image-classification-for-beginner/input?select=chest_xray
* Classes:

  * Normal
  * Pneumonia
